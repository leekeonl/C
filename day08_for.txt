     for 문 
      : 지정한 조건을 만족 할 때, 조건에 해당하는 명령문(블록)을 지정한 횟수만큼 반복 실행
        for문은..(시작시 할 일 ; 조건문 ; 반복이 끝나고 할 일) 을 한번에 설정 가능하고,
        주로 반복 횟수가 일정 할 때 사용.
	
	
	ex ) 사용법 및 실행 순서

		for( 1.초기식 ; 2.조건식 ; 4.증감식 ){
			3.명령문( 블록 );
		}	
		
		1. 초기식 : 시작시 할 일이고, 한 번만 실행.
			    반복을 위한 변수의 선언 및 초기화에 사용(생략 가능).
		2. 조건식 : 반복여부 결정(조건문).

			    조건식을 생략하면 조건은 항상 참.
		3. 명령문 : 조건식이 참일때 반복 할 문장.
			    여러개의 문장을 반복하고자 할 때에는 블록{ }으로 지정.
		4. 증감식 : 반복이 끝나고 진행되며, 조건식의 변화를 주기 위해 사용
			    반복의 조건을 '거짓'으로 만드는 증가 및 감소연산.

	        실행순서 : 1 -> 2 -> 3 -> 4 -> 2 -> 3 -> 4 ->...-> 2 조건이 거짓이면 종료

	ex ) 사용법 및 실행 순서
		
		for( 1.초기식 ; 2.조건식 ; 9.증감식){
			3.명령문 1-1;
		    	
			for( 4.초기식 ; 5.조건식 ; 7.증감식 ){
				6.명령문 2;
		   	} // 안쪽 for문
			8.명령문 1-2;
		} //바깥쪽 for문			 
   		
	실행순서 : 1 -> 2 -> 3 -> 4 -> 5 -> 6 -> 7 ->...-> 5 -> 6 -> 7 -> 5가 거짓 -> 8 -> 9 ->
			2 -> 3 -> 4 -> 5 -> 6 -> 7 ->...-> 5 -> 6 -> 7 -> 5가 거짓 -> 8 -> 9 ->
			2 -> 3 -> 4 -> 5 -> 6 -> 7 ->...-> 5 -> 6 -> 7 -> 5가 거짓 -> 8 -> 9 ->
			2가 거짓이면 종료  
     */
